# Diversões Maracanã — Landing Page

Site vitrine single-page do parque em Niterói — RJ.

## Seções

1. **Header** — nome em destaque + menu (Início, Nossa História, Atrações, Contato)
2. **Hero** — título, subtítulo e CTA «Planeje sua visita»
3. **Nossa História** — texto desde 1969
4. **Atrações** — grid com 5 brinquedos e placeholders de imagem
5. **Footer** — frase de encerramento, localização, redes e copyright 1969–2026
